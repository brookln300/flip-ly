### 2. Three New Redis-Powered Features (Brainstormed, Built, Tested, Audited)

**Search Replay ("Since You Last Searched")**
- Tracks every search query per user (Redis sorted sets + hashes)
- On return visit, diffs current results vs stored baseline to surface new matches
- Files: `app/lib/redis.ts` (hashQuery, storeSearchResults, getNewMatches, recordSearchQuery, getTopSearches), `app/api/listings/replay/route.ts` (NEW), `app/api/listings/route.ts` (fire-and-forget tracking added)

**Price Bands (Price Intelligence)**
- Nightly cron computes p25/median/p75 per market+tag from listing prices
- Search results annotated with price position (exceptional/good/average/above)
- Files: `app/lib/redis.ts` (PriceBand interface, setPriceBand, getPriceBand, getAllPriceBands, getPricePosition), `app/api/cron/price-bands/route.ts` (NEW), `app/api/listings/price-context/route.ts` (NEW), `app/api/listings/route.ts` (price context annotation)

**Resale Radar (High-Flip-Potential Feed)**
- Flags listings with resale potential during AI enrichment
- Bi-hourly cron caches flagged listings per market in Redis (2hr TTL)
- Files: `app/lib/redis.ts` (setResaleRadar, getResaleRadar), `app/api/cron/resale-radar/route.ts` (NEW), `app/api/listings/resale-radar/route.ts` (NEW), `app/lib/ai/enrich-listing.ts` (added resale_flag to DB write), `supabase/migrations/20260411_add_resale_flag.sql` (NEW — applied to prod)
