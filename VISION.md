# flip-ly — Vision

## What It Is
flip-ly is an AI-powered local deal discovery platform. Users find undervalued items they can resell for profit, surfaced and scored by AI across local listing sources.

## Current Phase
**Post-launch optimization.** The product is live. Work now centers on making the existing experience faster, more reliable, and easier to onboard into — not on net-new feature surface area.

### Focus Areas
- **Performance** — drive page load and interaction speed down.
- **Mobile bug fixes** — close the gap between desktop and mobile quality.
- **Onboarding flow** — get a new user to their first evaluated deal quickly.
- **Analytics setup** — instrument the funnel so optimization is measured, not guessed.

## Success Criteria
| Metric | Target |
|--------|--------|
| Page load | Under 2 seconds |
| Responsive design | Mobile-first across all viewports |
| Deal accuracy | Above 85% |
| Time to find + evaluate a deal | Under 60 seconds |

## Non-Goals (For Now)
These are explicitly out of scope during the optimization phase. Parked, not rejected.
- Social features
- Native mobile app
- International expansion
